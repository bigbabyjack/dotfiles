# Project Overview

## Purpose
This is a cross-platform dotfiles configuration repository for macOS and Arch Linux with profile support. It manages system configurations, development tools, and application settings using GNU Stow for symlink management.

## Main Applications Configured
- **Neovim**: Lua-based configuration with kickstart.nvim, LSP, and extensive plugins
- **Hyprland**: Wayland compositor for Arch Linux with idle, lock, and wallpaper configs
- **Tmux**: Terminal multiplexer with custom scripts
- **Zsh**: Shell configuration with Starship prompt
- **Ghostty**: Terminal emulator
- **Waybar**: Status bar for Wayland
- **Wofi**: Application launcher
- **Dunst**: Notification daemon
- **Fastfetch**: System information tool

## Target Platforms
- Arch Linux (primary - currently active)
- macOS (supported but not primary)
