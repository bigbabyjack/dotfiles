# Tech Stack

## Programming Languages
- **Lua**: Primary language for Neovim configuration (detected by Serena)
- **Shell Script**: Bash/Zsh for utility scripts and setup
- **Python**: Development environment support
- **Go**: Development environment support
- **Rust**: Development environment support
- **Node.js/TypeScript**: Development environment support

## Configuration File Formats
- TOML: StyLua configuration, various app configs
- YAML: Potential use in various configurations
- INI-style: Hyprland configurations
- Lua: Neovim and potentially WezTerm configs

## Key Tools & Utilities
### Window Management (Arch Linux)
- Hyprland (Wayland compositor)
- Waybar (status bar)
- Wofi (launcher)
- Dunst (notifications)

### Development Tools
- Neovim (editor)
- Git (version control)
- Lazygit (Git TUI)
- LSP servers (for various languages)
- StyLua (Lua formatter)

### Shell & Terminal
- Zsh (shell)
- Starship (prompt)
- Tmux (multiplexer)
- Ghostty (terminal emulator)

### CLI Utilities
- eza (modern ls)
- fd (modern find)
- ripgrep (modern grep)
- btop (system monitor)
- fastfetch (system info)

## Package Management
- **Arch Linux**: pacman, yay (AUR helper)
- **macOS**: Homebrew (documented but not primary)
- **Neovim**: lazy.nvim (plugin manager)
